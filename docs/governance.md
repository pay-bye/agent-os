# Governance

The root `GOVERNANCE.md` file defines maintainer authority for Agent OS.

Compatibility ownership stays in this repository:

- invocation contract: `contracts/invocation/v1.openapi.yaml`
- release schema: `contracts/release/v1.schema.json`
- vocabulary schema: `contracts/vocabulary/v1.schema.json`
- release artifacts and attestations published from `github.com/pay-bye/agent-os`

The future `github.com/pay-bye/agent-os-catalog` repository is discovery only. It may link to this
material, but it does not supersede or override component-owned truth.
